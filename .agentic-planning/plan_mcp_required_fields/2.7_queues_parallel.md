# 2.7 Queues API - Добавление обязательного `fields` (Parallel)

**Тип:** Parallel | **Приоритет:** Средний | **Tools:** 6

---

## Tools

1. **create_queue** - создание (возвращает Queue)
2. **get_queue** - получение одной (возвращает Queue)
3. **get_queues** - список (возвращает Queue[])
4. **get_queue_fields** - поля очереди (возвращает Field[])
5. **manage_queue_access** - доступ (возвращает permissions object)
6. **update_queue** - обновление (возвращает Queue)

---

## Изменения

### create_queue
- **Schema:** `src/tools/api/queues/create-queue.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/queues/create-queue.tool.ts` → `ResponseFieldFilter.filter<QueueWithUnknownFields>(queue, fields)`

### get_queue
- **Schema:** `src/tools/api/queues/get-queue.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/queues/get-queue.tool.ts` → `ResponseFieldFilter.filter(queue, fields)`

### get_queues
- **Schema:** `src/tools/api/queues/get-queues.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/queues/get-queues.tool.ts` → `queues.map(q => ResponseFieldFilter.filter(q, fields))`

### get_queue_fields
- **Schema:** `src/tools/api/queues/get-queue-fields.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/queues/get-queue-fields.tool.ts` → `queueFields.map(f => ResponseFieldFilter.filter(f, fields))`

### manage_queue_access
- **Schema:** `src/tools/api/queues/manage-queue-access.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/queues/manage-queue-access.tool.ts` → `ResponseFieldFilter.filter(permissions, fields)`

### update_queue
- **Schema:** `src/tools/api/queues/update-queue.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/queues/update-queue.tool.ts` → `ResponseFieldFilter.filter(queue, fields)`
