# 2.5 Links API - Добавление обязательного `fields` (Parallel)

**Тип:** Parallel | **Приоритет:** Критический | **Tools:** 2

---

## Tools

1. **create_link** - создание связи (возвращает Link)
2. **get_issue_links** - список связей (возвращает Link[])

**Не требует:** `delete_link` (void)

---

## Особенности

**ВАЖНО:** В `get_issue_links.tool.ts` есть ручная фильтрация полей (строки 67-92) - **УДАЛИТЬ** и заменить на `ResponseFieldFilter`.

---

## Изменения

### create_link
- **Schema:** `src/tools/api/issues/links/create/create-link.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/issues/links/create/create-link.tool.ts` → `ResponseFieldFilter.filter<LinkWithUnknownFields>(link, fields)`
- **Tests:** Добавить `fields: ['id', 'type', 'object']`

### get_issue_links
- **Schema:** `src/tools/api/issues/links/get/get-issue-links.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/issues/links/get/get-issue-links.tool.ts`
  - **УДАЛИТЬ ручную фильтрацию** (строки 67-92)
  - **ДОБАВИТЬ:** `links.map(link => ResponseFieldFilter.filter(link, fields))`
- **Tests:** Добавить `fields`
