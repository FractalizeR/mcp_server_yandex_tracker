# 2.6 Projects API - Добавление обязательного `fields` (Parallel)

**Тип:** Parallel | **Приоритет:** Средний | **Tools:** 4

---

## Tools

1. **create_project** - создание (возвращает Project)
2. **get_project** - получение одного (возвращает Project)
3. **get_projects** - список (возвращает Project[])
4. **update_project** - обновление (возвращает Project)

**Не требует:** `delete_project` (void)

---

## Изменения

### create_project
- **Schema:** `src/tools/api/projects/create-project.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/projects/create-project.tool.ts` → `ResponseFieldFilter.filter<ProjectWithUnknownFields>(project, fields)`
- **Tests:** Добавить `fields: ['id', 'name']`

### get_project
- **Schema:** `src/tools/api/projects/get-project.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/projects/get-project.tool.ts` → `ResponseFieldFilter.filter(project, fields)`
- **Tests:** Добавить `fields`

### get_projects
- **Schema:** `src/tools/api/projects/get-projects.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/projects/get-projects.tool.ts` → `result.projects.map(p => ResponseFieldFilter.filter(p, fields))`
- **Tests:** Добавить `fields`

### update_project
- **Schema:** `src/tools/api/projects/update-project.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/projects/update-project.tool.ts` → `ResponseFieldFilter.filter(project, fields)`
