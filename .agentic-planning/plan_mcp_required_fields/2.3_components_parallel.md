# 2.3 Components API - Добавление обязательного `fields` (Parallel)

**Тип:** Parallel | **Приоритет:** Высокий | **Tools:** 3

---

## Tools

1. **create_component** - создание (возвращает Component)
2. **get_components** - список (возвращает Component[])
3. **update_component** - обновление (возвращает Component)

**Не требует:** `delete_component` (void)

---

## Изменения

### create_component
- **Schema:** `src/tools/api/components/create-component.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/components/create-component.tool.ts` → `ResponseFieldFilter.filter<ComponentWithUnknownFields>(component, fields)`
- **Tests:** Добавить `fields: ['id', 'name']`

### get_components
- **Schema:** `src/tools/api/components/get-components.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/components/get-components.tool.ts` → `components.map(c => ResponseFieldFilter.filter(c, fields))`
- **Tests:** Добавить `fields`

### update_component
- **Schema:** `src/tools/api/components/update-component.schema.ts` → `fields: FieldsSchema`
- **Tool:** `src/tools/api/components/update-component.tool.ts` → `ResponseFieldFilter.filter(component, fields)`
- **Tests:** Добавить `fields`
